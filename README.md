# 🧮 Calculadora Científica

Calculadora científica desarrollada con **Python**, que incluye historial de operaciones, limpieza del historial y funciones matemáticas esenciales.

---

## 🚀 Tecnologías

- Python

---

## ✨ Características

- 🧮 Operaciones matemáticas básicas
- 📚 Historial de cálculos
- 🗑️ Opción para limpiar el historial
- 📐 Funciones científicas esenciales
- 💻 Interfaz intuitiva y fácil de usar

---

## ▶️ Cómo ejecutar

1. Clona este repositorio.
2. Instala las dependencias (si aplica):

```bash
pip install -r requirements.txt
```

3. Ejecuta:

```bash
python Calculadora.py
```
